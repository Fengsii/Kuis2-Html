Perkembangan dan Versi HTML

html punya beberapa versi dari versi yang paling tua hingga yang terbaru

  1. HTML 1.0(Juni 1993) adalah versi HTML pertama, namun tidak resmi mrilis
  2. HTML 2.0(24 November 1995) adalah versi HTML kedua yang resmi pertamakali beredar di pasar dan rilis oleh IETF
  3. HTML 3.0(28 Maret 1995) versi ini gagal beredar, karena banyak perubahan yang memicu perdebatan
  4. HTML 3.2(14 Januari 1997) versi resmi dirilis W3C pertamkali
  5. HTML 4.0(24 April 1998) versi pengembangan dari yang sebelumnya
  6. HTML 4.01(24 Desember 1999) versi perbaikan dari HTML 4.0
  7. HTML 1.0(26 Januari 2000) pengembangan dari HTMl 4.01 dengan mengadopsi XML
  8. HTML 2.0(Agustus 2002--Juli 2006) versi kedua dari XHTML
  9. HTML 5(28 Oktober 2014) versi HTML saat ini
